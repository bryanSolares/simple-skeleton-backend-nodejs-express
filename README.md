# Simple Skeleton for API NodeJS and Express

## Commands

-   Start application with Node

```
npm run start
```

-   Start application in mode development with Nodemon

```
npm run dev
```

-   Format files with prettier

```
npm run format
```

-   Fixed error linter with ESlint

```
npm run linter
```

-   Test code

```
npm run test
```
